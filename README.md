Sublime snippets
================

A collection of [sublime text](http://www.sublimetext.com/3) snippets

*   JavaScript
    * func -> basic function
    * desc -> bdd style describe block
    * it   -> bbd style it block
    * Qunit
        * module
        * test
    * m (A JS library written by [rudenoise](https://github.com/rudenoise))
        * block
        * module
*   PHP
    * PHPUnit
        * testCase (pu-testCase)
        * test (pu-test)
        * assertEquals (pu-ae)
